NOMAD Installations
We continuesly improve the software behind NOMAD and NOMAD Oasis. NOMAD operates different installations to try features that are currently developed, have an empty test sandbox, or access older version of NOMAD.

Beta
This installation runs the newest version of the NOMAD software and uses the official NOMAD data. It is called beta or staging version.
  • Experimental features activated.
  • Shows the actual NOMAD data.
  • Frequently updated.

This installation runs the official released version of the NOMAD software, but on data that is routinly wiped.
  • Official NOMAD software.

  • Can be used for testing.

Legacy
This installation runs an old version of NOMAD (v0). Newer versions use a different API that is incompatible.
  • Runs The old v0 NOMAD version.
  • Frozen, data upto end of 2021.
  • Offers the legacy NOMAD API.
