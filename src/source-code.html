Source code repositories
The NOMAD software consists of many parts and is maintained in several source code repositories.
vpn_lock
GitLab@MPCDF

The main git repository is a GitLab project called nomad-fair and is hosted in MPCDFs GitLab instance. This is used by the FAIRmat and NOMAD team members.

Here we coordinate the intrnal development work, keep out backlog, and run all CI/CD. The project is openly readable, but you need an MPCDF account for active participation.

public
GitHub

This is a public version of the main repository hosted at GitHub. It contains the latest develop branch. This is aimed at the wider community.

Everyone can post and dicsuss issues and can contribute via pull requests and the usual GitHub flow.

Parsers at GitHub

Some components, especially all the parsers, are mainted in seperate git projects. Those are all publically hosted at GitHub in the nomace-coe GitHub organisation.
