NOMAD
Share data, not files.
NOMAD lets you extract and publish structured data with rich meta-data. It provides a unified way to find, access, interoperate with, and re-use (FAIR) millions of data from different codes and sources.

What is NOMAD?
It is a free web-service that let's you share you data or use comprehensive data that others provide. You can use NOMAD to organise, analyse, share, and publish your materials science data, as well as explore, download, and analyse your colleagues data.

FAIR data
Why is NOMAD different?
NOMAD is not about sharing files, it is about sharing data and making data FAIR. All data is organized in well-defined structures that are described by a formal schema. This is the basis for search (findable), APIs (accessible), converting data (interoperable), and running tools (re-usable).
Structured data and Metainfo
All data in NOMAD follows a schema that defines how data is structured and cross-references in hierarchies. This schema is called the NOMAD Metainfo. It defines a general domain independent super structure, as well as highly detailed specialized data from specific methods, tools, and programs.
  • Use NOMAD's parsers to automatically structure your data.
  • Use and extend the existing NOMAD Schema.
  • All data hierarchically organized and connected.
Analysis tools
Since all data is uniformly organized, you can use existing tools and build your tools own to analyse data from various sources. The NOMAD Remote Tools Hub (NORTH) allows you to run containerized tools and Jupyternotebooks directly on your data.
  • Run Jupyterlab to access and author your uploads in NOMAD.
  • Predefined notebooks for machine learning in the AI Toolkit.
  • All data available via API. Analyse your data from everywhere.
