          NOMAD
          Share data, not files.
          NOMAD lets you extract and publish structured data with rich meta-data. It provides a unified way to Find, Access, Interoperate with, and Re-use millions of FAIR data from different codes and sources.
          Features
          What is NOMAD?
          NOMAD is a free web-service that lets you share your data or use comprehensive data that others provide. You can use NOMAD to organise, analyse, share, and publish your materials science data, as well as explore, download, and analyse your colleagues data.
          cloud_upload
          Drag'n'Drop
          NOMAD organises your files as you would on your hard-drive. Upload file-by-file or zip and upload whole directory structures.
          link
          Get a DOI
          NOMAD allows you to publish and archive your data for free. Assign a DOI to uploaded datasets and reference your papers.

          FAIRmat
          Join the FAIRmat consortium and revolutionize the way research data is managed by embracing FAIR data. FAIRmat develops and operates NOMAD to build a federated data infrastructure for materials science.
          FAIR data
          Why is NOMAD different?
          NOMAD is not about sharing files, it is about sharing data and making data FAIR. All data is organized in well-defined structures that are described by a formal schema. This is the basis for search (Findable), APIs (Accessible), converting data (Interoperable), and running tools (Re-usable).
          Structured data and Metainfo
          All data in NOMAD follows a schema that defines how data is structured and cross-references in hierarchies. This schema is called the NOMAD Metainfo. It defines a general domain independent super structure, as well as highly detailed specialized data from specific methods, tools, and programs.
          • NOMAD's parsers automatically structure your data
          • You can extend the existing NOMAD Schema
          • All data hierarchically organized and connected
          Analysis tools
          Since all data is uniformly organized, you can use existing tools and build your tools own to analyse data from various sources. The NOMAD Remote Tools Hub (NORTH) allows you to run containerized tools and Jupyternotebooks directly on your data.
          • Jupyterlab to access and modify files in NOMAD
          • Predefined notebooks for machine learning in the AI Toolkit
          • All data available via API, analyses data from everywhere

          Other NOMAD versions
          We operate different installations with differnet NOMAD versions. You can try features that are currently developed, have an empty test sandbox, or access older version of NOMAD.

          Beta
          This installation runs the newest version of the NOMAD software and uses the official NOMAD data. This installation is called "beta" or "staging".

          • Shows the actual NOMAD data
          • Frequently updated

          This installation runs the official released version of the NOMAD software, but on data that is routinly wiped.
