NOMAD
Share data, not files.
NOMAD processes your files to extract and publish structured data with rich meta-data. We provide a unified way to find, access, interoperate with, and re-use (FAIR) millions of data from different codes and sources.
Features
What is NOMAD?
It is a free web-service that let's you share you data or use comprehensive data that others provide. You can use NOMAD to organise, analyse, share, and publish your materials science data, as well as explore, download, and analyse your colleagues data.
link
Get a DOI
NOMAD allows you to publish and archive your data for free. Assign a DOI to uploaded datasets and reference your papers.
code
All data at a glance
NOMAD extracts structured, machine-actionable data from over 50 of formats and codes. See all Your data at a glance and have it prepared for analysis.
construction
Ready to use Tools
Run Jupyter notebooks and other tools directly on NOMAD. Directly access your data and publish your analysis code. No installation required.
cloud_upload
Drap'n'Drop
NOMAD organises your files as you would on your hard-drive. Upload file-by-file or zip and upload whole directory structures.
shield
You Control Your Data

extension
Extendable
Add your own file formats. Create custom ELNs. Create exploration dashboards. Build and run more analysis tools.
NOMAD is developed and used by FAIRmat to build a federated data infrastructure
The NFDI consortium FAIRmat uses NOMAD to connect data from many research groups, institutes, and universities to build a federated data infrastructure for materials science.
FAIR data
Why is NOMAD different?
NOMAD is not about sharing files, it is about sharing data and making data FAIR. All data is organized in well-defined structures that are described by a formal schema. This is the basis for search (findable), APIs (accessible), converting data (interoperable), and running tools (re-usable).
metainfo
NOMAD Metainfo
All data in NOMAD follows a schema that defines how data is organized, linked, and defined. This schema is called the NOMAD Metainfo. It can define general domain independent data, but also highly detailed specialized data from specific methods, tools, and programs.
  • Use NOMAD's parsers to automatically structure your data.
  • Use and extend the existing NOMAD Schema.
  • All data hierarchically organized and connected.
Analysis tools
Since all data is uniformly organized, you can use existing tools and build your own to analyse data from various sources. The NOMAD Remote Tools Hub (NORTH) allows you to run containerized tools and Jupyternotebooks directly on your data.
  • Run Jupyterlab to access and author your uploads in NOMAD.
  • Predefined notebooks for machine learning in the AI Toolkit.
  • All data available via API. Analyse your data from everywhere.
