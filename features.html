          Roadmap
          What features are coming next?
          We continuesly improve the software behind NOMAD and NOMAD Oasis. Here, you find the currently planned and developed features and if and where they can be already tried.
          Backlog
          These features are discussed or not yet merged. You can participate in the on our forum or open a new feature request.
          • Plug-ins
            Add schemas, ELNs, parsers, and normalizers to an Oasis installation without build your own Oasis image.
          • Oasis data transfer
            Allows to move uploads from one installation to another. E.g. publish data from your Oasis to the public NOMAD service.
          Develop
          These features are still developed and are available on the develop branch of our repository or Oasis using the develop image tag.
          • ElabFTW integration
            Import elabftw experiments into NOMAD via the .eln file format.
          • Hierarchical structures
            Certain complex systems (interfaces, metal organic frameworks) are broken down into sub-sections that can be visualized and accessed independently.
          In beta
          These features are still experimental and have been put on the NOMAD beta installation or Oasis using the latest image tag.
          • NORTH
            The NOMAD Remote Tools Hub allows to run containarized tools directly on your uploaded data.
          • Custom workflows and visualizer
            A new workflow model that allows you to create custom workflows via archive files, custom schema and ELNs, your own parsers.
          • Labfolder integration
            Import labfolder projects into NOAMD.
          Recently released
          These features have been recently released and can be used on the public NOMAD service or Oasis using the stable image tag.
          • Electronic Lab Notebooks
            Define structured data entry with schema's and NOMAD's ELN functionality.
          • Nexus support
            Allows you to upload nexus files and preview contents with H5Web.
          • Search apps
            Allows you to define custom search interfaces for your Oasis. Also includes curated apps on the public NOMAD service.
